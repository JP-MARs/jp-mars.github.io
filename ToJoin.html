How to join JP-MARs project

JP-MARsプロジェクトに参画したい方へ。

プロジェクト参加方法

現在、試運用中。ルール整備の途中です。以下、案。

  • ちょっとした修正などは、githubの機能（プルリクエスト、issue）等でご指摘ください。ただし、反応が非常に遅い可能性がありますのでご容赦ください。

  • 主要な開発メンバとして参画したい場合は、各リポジトリの責任者に何らかの方法で個別に連絡し、ご相談してください。

  • 新リポジトリの作成を希望する方は、JP-MARsプロジェクト発起人Grに何らかの方法で個別に連絡し、ご相談してください。

    • JP-MARs主要管理者：　XXX(YY大学)

※連絡先は本ページなどで公に公開しませんので、関係者をたどるなどしてコンタクトしてくださるよう、お願いいたします。



How to join JP-MARs

  • If you have any small modifications, please point them out to us via github functions (pull requests, issues, etc.). However, please note that the response may be very slow.

  • If you wish to participate as a key development member, please contact the person in charge of each repository individually in some way to discuss.

  • If you wish to create a new repository, please contact the JP-MARs project initiator, Gr, individually in some way and consult with him or her.
