[JP-MARsプロジェクト]について (About JP-MARs project)
------------------

| 日本の電磁界解析関係の研究者が主体となっている、オープンソースプロジェクトです。
| 特に、「日本向けに、ある程度ソースコードを読みやすく、実用的にも教育的にも役に立つ基本OSS」を提供します。日本の磁界解析関係研究者が発起人ですので、電気系・電磁気系のソースコードが多いですが、汎用的なものも含めて、様々なものを公開して行きたいと思います。
| This project is an open source project led by researchers related to electromagnetic field analysis in Japan. In particular, we provide "basic OSS mainly for Japanese that would be easy to read source code and useful for both practical and educational purposes.


公開済みOSS一覧 (OSS list)
========

* **SparseSolv** (https://github.com/JP-MARs/SparseSolv)　大規模疎行列とそのソルバを提供します。磁界解析に特に有効とされる「加速係数付きICCG法」を提供している珍しいOSSかもしれません。(It provides a large sparse matrix and its solver. It may be a rare OSS that provides the “ICCG method with acceleration factors,” which is particularly useful for magnetic field analysis.)
