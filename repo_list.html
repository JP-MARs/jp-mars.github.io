SparseSolv
SparseMatrix and its solvers including shifted-ICCG

Main contributer: Takahiro Sato (Muroran institute of technology)

repository is here!

 疎行列とその線形ソルバを提供するOSSです。FEMなどで必要な大規模疎行列を効果的に扱えます。電磁気のFEMに特有の特異行列を解く際に有効な加速係数付きICCG法を提供しています。
 疎行列クラスは、C++用の行列OSSであるEigenのラッパとして定義されています。線形ソルバは独自実装のICCG法やMRTR法のほか、EigenやIntelコンパイラのソルバのラッパー機能もあります。
 日本の磁界系研究者が開発したソルバの有用性を世界に発信することを意識しています

 This OSS provides sparse matrices and their linear solvers. It effectively handles large sparse matrices required for FEM and similar applications. It includes a shifted-ICCG method with acceleration factors, particularly effective for solving singular matrices specific to electromagnetic FEM.
 The sparse matrix class is defined as a wrapper for Eigen, a C++ matrix OSS. The linear solver includes oroginal the ICCG and MRTR methods, as well as wrapper functionality for Eigen and Intel Compiler solvers.

MF-MeshConverter
Mesh converter with modification functions

Main contributer: Takahiro Sato (Muroran institute of technology)

repository is here!

 メッシュの形式変換と、メッシュ修正機能を統合したコマンドラインソフトウェアです。節点修正、材料ID変更、2D→3D積み上げなどの、メッシュを操作する処理をJSONで記述することで実施できます。実行形式とlib形式があり、単独使用や自作ソフトへの組み込み、どちらも可能です。

 This command-line software integrates mesh format conversion and mesh modification functions. It enables node modification, material ID changes, 2D->3D stacking, and other mesh manipulation processes by describing them in JSON. Available in executable and library formats, it can be used standalone or embedded into custom software.
